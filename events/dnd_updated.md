# dnd_updated event

    {
        "type": "dnd_updated",
        "user": "U1234",
        "dnd_status": {
            "dnd_enabled": true,
            "next_dnd_start_ts": 1450387800,
            "next_dnd_end_ts": 1450423800,
            "snooze_enabled": true,
            "snooze_endtime": 1450373897
        }
    }

The `dnd_updated` event is sent to the current user when their Do Not Disturb settings have changed.
